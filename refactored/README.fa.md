# وُکسِل‌فورج (VoxelForge) — سامانه‌ی سفارش چاپ سه‌بعدی

سامانه‌ی کامل و **کاملاً آفلاین** برای سفارش آنلاین چاپ سه‌بعدی:
آپلود فایل **STL**، **اسلایس واقع‌گرا در مرورگر**، پیش‌نمایش سه‌بعدی با هایلایت اورهنگ،
قیمت‌گذاری زنده، پرداخت **کارت‌به‌کارت** ایرانی، فاکتور قابل‌چاپ، و پنل مدیریت کامل.
رابط کاربری کاملاً **فارسی** و **راست‌چین**.

---

## 🚀 شروع سریع

```bash
cd refactored
npm install
npm run dev                 # حالت توسعه روی http://localhost:3000
# یا برای پروداکشن:
npm run build
node dist/server/server.js
```

پس از اجرا، به `/auth` بروید و ثبت‌نام کنید. **اولین حسابی که ساخته شود، به‌طور خودکار مدیر کارگاه است.**
سپس در `/settings` قیمت و اطلاعات کارت بانکی خود را وارد کنید.

> 🔌 **بدون اینترنت کار می‌کند.** هیچ سرویس ابری‌ای لازم نیست؛ همه‌ی داده‌ها و فایل‌ها
> در پوشه‌ی `data/` ذخیره می‌شوند. برای پشتیبان‌گیری همان پوشه را کپی کنید یا از دکمه‌ی
> «دانلود پشتیبان» در صفحه‌ی تنظیمات استفاده کنید.

## ✨ امکانات

- **موتور اسلایس واقع‌گرا** — وزن مثل Cura/PrusaSlicer از روی پوسته، اینفیل و ساپورت محاسبه می‌شود (±۱۰٪).
- **پیش‌نمایش سه‌بعدی** بدون وابستگی خارجی، با چرخش ماوس و **هایلایت قرمز اورهنگ‌ها**.
- **چرخش و مقیاس زنده** — مدل را بچرخانید/بزرگ‌کوچک کنید و قیمت/ساپورت آنی بازمحاسبه شود.
- **چند متریال** (PLA/PETG/ABS/TPU/رزین) با چگالی و ضریب قیمت متفاوت.
- **پریست‌های کیفیت** (پیش‌نویس / استاندارد / دقیق).
- **تعداد، رنگ، و حداقل مبلغ سفارش**.
- **هشدار حجم چاپ** وقتی مدل در بستر پرینتر جا نمی‌شود.
- **احراز هویت محلی** (scrypt + کوکی نشست) — اولین کاربر = مدیر.
- **پرداخت کارت‌به‌کارت** با کارت بانکی گرافیکی و کپی یک‌کلیکی.
- **پنل مدیریت**: داشبورد آماری + نمودار درآمد، جست‌وجو، صفحه‌بندی، خروجی CSV.
- **فاکتور قابل‌چاپ** (Print-to-PDF) و **پشتیبان‌گیری/بازیابی کامل داده**.
- **یکپارچگی قیمت** — قیمت روی سرور بازمحاسبه می‌شود و غیرقابل دستکاری است.
- **افزونه‌ی کروم** — روی MakerWorld/Thingiverse/Printables قیمت آنی + سفارش یک‌کلیکی ([راهنما](./extension/README.md)).

## ⚙️ پیکربندی

دو راه برای تنظیم کسب‌وکار:

1. **از پنل ادمین** (توصیه‌شده): صفحه‌ی `/settings` — قیمت هر گرم، حداقل سفارش، حجم چاپ،
   شماره کارت، شبا و اطلاعات تماس. این مقادیر در `data/voxelforge.json` ذخیره می‌شوند.
2. **مقادیر پیش‌فرض اولیه** در `src/lib/business.ts` (نام، شماره کارت پیش‌فرض و …) که فقط
   هنگام نخستین اجرا به‌عنوان مقدار اولیه استفاده می‌شوند.

## 🗂 ساختار پروژه (خلاصه)

```
refactored/
├── src/
│   ├── routes/            # صفحات: index, quote, orders, admin, settings, profile, invoice, auth
│   ├── lib/
│   │   ├── stl-parser.ts          # موتور اسلایس + قیمت‌گذاری + فرمت فارسی
│   │   ├── *.functions.ts         # توابع سرور (auth, orders, admin, settings, backup, profile)
│   │   ├── types.ts               # تایپ‌های مشترک کلاینت/سرور
│   │   └── server/                # بک‌اند محلی (store, session, files, map)
│   ├── components/        # Navbar, Footer, StlViewer
│   └── hooks/             # use-auth, use-settings
├── scripts/selftest.ts   # تست خودکار موتور اسلایس
├── docs/                 # این مستندات
└── data/                 # دیتابیس و فایل‌ها (runtime، در git نیست)
```

جزئیات کامل در [معماری](./docs/ARCHITECTURE.md).
