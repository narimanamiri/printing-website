# وُکسِل‌فورج (VoxelForge) — سامانه سفارش چاپ سه‌بعدی

> این مخزن قبلاً شامل دو نمونه‌ی موازی بود. حالا به **یک اپلیکیشن واحد** یکپارچه شده
> که بهترین ویژگی‌های هر دو را به‌علاوه‌ی یک **موتور اسلایس واقع‌گرا** و **پیش‌نمایش سه‌بعدی**
> در خود دارد. کد در پوشه‌ی [`refactored/`](./refactored) قرار دارد.

سامانه‌ی سفارش آنلاین چاپ سه‌بعدی: آپلود فایل **STL**، **اسلایس واقعی در مرورگر**، قیمت‌گذاری زنده،
پرداخت **کارت‌به‌کارت** ایرانی، و پنل مدیریت صف چاپ. رابط کاربری کاملاً فارسی و راست‌چین.

```bash
cd refactored
npm install
npm run dev
```

پیکربندی و مستندات کامل: [`refactored/README.fa.md`](./refactored/README.fa.md)

## ✨ موتور اسلایس واقع‌گرا (نکته‌ی کلیدی)

برخلاف تخمین ساده‌ی `حجم × چگالی × اینفیل`، این نسخه مثل یک اسلایسر واقعی
(Cura / PrusaSlicer) وزن را تفکیک می‌کند:

- **مساحت سطح و جهت هر مثلث** از فایل STL استخراج می‌شود.
- **پوسته و دیواره‌ها**: روی سطوح عمودی، به ضخامت `تعداد دیواره × عرض خط`.
- **پوسته‌ی رویی/زیری**: روی سطوح افقی، به ضخامت `تعداد لایه × ارتفاع لایه`.
- **اینفیل**: فقط حجم توخالی داخلی، با درصد انتخابی پر می‌شود.
- **ساپورت**: تخمین حجم تکیه‌گاه برای اورهنگ‌های تند (با تشخیص خودکار نرمال‌ها).
- خروجی شامل **وزن، طول فیلامنت (متر)، زمان تخمینی چاپ و تفکیک گرم به گرم** است.

نتیجه: تخمین وزن معمولاً تا **±۱۰٪** با خروجی واقعی پرینتر می‌خواند، نه یک عدد دلخواه.

پریست‌های کیفیت (پیش‌نویس ۰٫۲۸، استاندارد ۰٫۲، دقیق ۰٫۱۲ میلی‌متر) و قیمت متفاوت
برای هر متریال (PLA/PETG/ABS/TPU/رزین) لحاظ شده است.

موتور در [`refactored/src/lib/stl-parser.ts`](./refactored/src/lib/stl-parser.ts).

## 🔒 یکپارچگی قیمت (امنیت)

قیمتی که در مرورگر دیده می‌شود فقط **پیش‌نمایش** است. هنگام ثبت سفارش،
تابع سرور [`createOrder`](./refactored/src/lib/orders.functions.ts) فایل STL را
**روی سرور دوباره اسلایس می‌کند** و قیمت معتبر را می‌نویسد. سیاست‌های RLS طوری سفت
شده‌اند که کاربر نتواند سفارش با قیمت دلخواه بسازد یا قیمت/وضعیت سفارش خود را دستکاری کند.

## 🧊 پیش‌نمایش سه‌بعدی

کامپوننت [`StlViewer`](./refactored/src/components/StlViewer.tsx) مدل را
بدون هیچ وابستگی خارجی (canvas خالص، با سایه‌زنی و چرخش با ماوس) رندر می‌کند.
