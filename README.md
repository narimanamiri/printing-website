# وُکسِل‌فورج (VoxelForge) — سامانه سفارش چاپ سه‌بعدی (کاملاً آفلاین)

> یک اپلیکیشن واحد در [`refactored/`](./refactored): آپلود **STL**، **اسلایس واقع‌گرا** در مرورگر،
> پیش‌نمایش سه‌بعدی، قیمت‌گذاری زنده، پرداخت **کارت‌به‌کارت** ایرانی و پنل مدیریت صف چاپ.
> رابط کاربری کاملاً فارسی و راست‌چین.

## ⚡ بدون نیاز به اینترنت

این نسخه هیچ سرویس ابری‌ای لازم ندارد. کل بک‌اند **محلی** است و حتی با اینترنت قطع کار می‌کند:

- **احراز هویت محلی** — رمز عبور با `scrypt` هش می‌شود، نشست با کوکی امن (`httpOnly`).
- **دیتابیس محلی** — یک فایل JSON در `refactored/data/` (بدون هیچ وابستگی native).
- **ذخیره فایل محلی** — فایل‌های STL و رسیدها روی دیسک در `refactored/data/`.
- **اولین حسابی که ساخته شود، به‌طور خودکار مدیر کارگاه است** — بدون تنظیم دستی.

```bash
cd refactored
npm install
npm run dev          # یا: npm run build && node dist/server/server.js
```

اولین بار که ثبت‌نام کنید مدیر می‌شوید و به `/admin` دسترسی دارید. تمام داده‌ها در
`refactored/data/` می‌مانند (این پوشه در git نادیده گرفته می‌شود؛ برای پشتیبان‌گیری همان را کپی کنید).

## ✨ موتور اسلایس واقع‌گرا

برخلاف تخمین ساده‌ی `حجم × چگالی × اینفیل`، وزن مثل یک اسلایسر واقعی (Cura/PrusaSlicer)
**تفکیک‌شده** محاسبه می‌شود — از روی **مساحت سطح و جهت هر مثلث** فایل STL:

- **پوسته و دیواره‌ها** روی سطوح عمودی، **پوسته‌ی رویی/زیری** روی سطوح افقی،
- **اینفیل** فقط حجم توخالی داخلی، **ساپورت** برای اورهنگ‌های تند،
- خروجی: **وزن، طول فیلامنت (متر)، زمان تخمینی چاپ و تفکیک گرم‌به‌گرم**.

تخمین وزن معمولاً تا **±۱۰٪** با خروجی واقعی پرینتر می‌خواند. موتور:
[`refactored/src/lib/stl-parser.ts`](./refactored/src/lib/stl-parser.ts).
تست خودکار: `cd refactored && node --experimental-strip-types scripts/selftest.ts`.

## 🔒 یکپارچگی قیمت

قیمت مرورگر فقط **پیش‌نمایش** است. هنگام ثبت سفارش، تابع سرور
[`createOrder`](./refactored/src/lib/orders.functions.ts) فایل STL را **روی سرور دوباره اسلایس
می‌کند** و قیمت معتبر را می‌نویسد؛ کاربر نمی‌تواند قیمت را دستکاری کند.

## 🧊 پیش‌نمایش سه‌بعدی

[`StlViewer`](./refactored/src/components/StlViewer.tsx) مدل را بدون هیچ وابستگی خارجی
(canvas خالص، سایه‌زنی و چرخش با ماوس) رندر می‌کند.

## ⚙️ پیکربندی کسب‌وکار

قیمت هر گرم، حداقل سفارش، حجم چاپ و اطلاعات کارت بانکی را از **پنل ادمین** (`/settings`)
تنظیم کنید — بدون نیاز به ویرایش کد. مقادیر اولیه در
[`refactored/src/lib/business.ts`](./refactored/src/lib/business.ts).
